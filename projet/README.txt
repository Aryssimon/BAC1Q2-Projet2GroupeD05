Pour exécuter le projet:
1) installer python, site officiel: https://www.python.org/downloads/

2) installer le module flask via l'invite de commande : "pip install flask"

3) placer la base de données dans ce dossier-ci (avec ce README et les fichiers pythons)
   !! celle-ci doit porter le nom 'inginious.sqlite' sans les guillemets.

4) run le fichier flask_file.py avec python (*)

5) ouvrir un navigateur internet

6) entrer l'url suivante: "localhost:5000"


(*)Par exemple dans l'invite de commande, en vous dirigeant à l'aide de "cd" jusque dans ce dossier,
ensuite faire la commande "python flask_file.py"
